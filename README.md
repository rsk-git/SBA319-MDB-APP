Project Name: SBA319-MDB

Description: This project is a server application with Node, Express and MongoDB mongoose. It provides CRUD operations for posts, comments and users.

Technologies used: Node.js
Express
MongOB
mongoose
dotenv
morgan
cors

Installation: npm install, express, morgan, cors, mongoose and dependencies.

Routes -
posts - CRUD operations
users- CRUD operations 
comments - CRUD operations 

Dummy data is created in a seperate folder named data for posts, users and comments which can be used for API testing.
